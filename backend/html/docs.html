API Documentation

Base URL: https://kathut.de/api

Endpoint: /badge

Description: Generates a badge image in SVG format.

Method: GET

Parameters:

Parameter Description
label Label text for the badge.
message Message text for the badge.
labelColor Color of the label text. Default: grey.
color Background color of the badge. Default: lightgrey.
style Style of the badge. Default: flat.
logo Logo name for the badge.
logoColor Color of the logo. Default: #FFFFFF.
linkLeft Left link URL for the badge.
linkRight Right link URL for the badge.
Feather Icons must be prefixed with ft (ie: ftgithub)
Simple Icons must be prefixed with si (ie: siGithub)
Octicons must be prefixed with oct (ie: octmark-github)
Slug Viewer

Response:

Content-Type: image/svg+xml

Body: SVG image data representing the generated badge.

Endpoint: /id

Description: Generates a unique identifier.

Method: GET

Response:

Body: JSON object containing a unique generated URL safe identifier.

{ "id": "The generated unique identifier" }

Endpoint: /cat

Endpoint: /tod

Description: Generates a truth and dare.

Method: GET

Response:

Body: JSON object containing a truth and dare.

{ "truth": "The generated truth", dare: "The generated dare" }

Endpoint: /headers

Description: Responds with every header the client sends.

Method: GET

Response:

Body: JSON object containing all headers.

{ "dnt": "1", "sec-gpc": "1", "connection": "keep-alive", "upgrade-insecure-requests": "1" }
